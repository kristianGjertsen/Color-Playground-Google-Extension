# Color-Playground-Google-Extension

Bruk tilpassbare fargepaletter på nettsider, med egne forhåndsinnstillinger per side, AI-genererte paletter og kontroll på elementnivå.

Endre utseendet på hvilken som helst nettside umiddelbart med smarte fargetemaer. Tilpass paletter, oppdag automatisk eksisterende farger på siden, og generer nye temaer ved hjelp av AI.

*Fungerer på mange elementer på mange sider, men ikke alle elementer kan nås og injectes med css korrekt
